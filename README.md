# Suma sencilla

Repositorio con un script Python muy simple que pide dos números al usuario y muestra su suma.

## Archivos
- `codigo.py` — Script Python con las funciones:
  - `sumar(a, b)` — devuelve la suma de a y b.
  - `pedir_numero(mensaje)` — solicita un número al usuario y valida la entrada.

## Requisitos
- Python 3

## Uso
1. Abrir terminal en la carpeta del proyecto:
   ```
   cd c:\Users\JM\Documents\GitHub\CLASE_11_06_26
   ```
2. (Opcional) Renombrar `codigo.txt` a `codigo.py`:
   ```
   ren codigo.txt codigo.py
   ```
3. Ejecutar:
   ```
   python codigo.py
   ```
4. Seguir las instrucciones en pantalla. Ejemplo:
   ```
   Suma sencilla
   Ingresa el primer número: 2
   Ingresa el segundo número: 3
   Resultado: 5.0
   ```

## Nota
El archivo actual está con extensión `.txt`; es recomendable cambiarla a `.py` para ejecución directa.
